# RAG Data Assistant

Portfolio project: a Retrieval-Augmented Generation (RAG) assistant to help discover and understand data assets using natural language.

## Project goal
Help new analysts find which tables, columns and metrics exist, where they are, and how to use them — based only on official documentation.

## Tech stack
- Python
- LlamaIndex
- OpenAI
- ChromaDB
- Streamlit

## Status
Work in progress — initial setup and documentation phase.
